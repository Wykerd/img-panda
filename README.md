# Image Panda
## An image utility engine for content identification and manipulation

# Features

- Content ID

# Dependencies

## Must be installed seperately
- OpenSSL
- libuv
- OpenCV

## Compiled from submodules
- lexbor
- llhttp
- simdjson

## Toolchain
- `CMake` 3.11 or later
- `gcc` version supporting c11 and c++14
- `NodeJS` (for llhttp)
- `npm` (for llhttp)
